# 🎬 Desafio SQL - Banco de Dados de Filmes | DIO 🚀

Este projeto é parte de um desafio da **DIO (Digital Innovation One)**, no qual são propostas consultas SQL para serem aplicadas em um **banco de dados relacional sobre filmes**. O objetivo é exercitar conceitos como filtros, junções, ordenações, agrupamentos e uso de funções agregadas em SQL.

> 💡 **Dica**: Para visualizar e executar os scripts corretamente, recomenda-se utilizar o **SQL Server Management Studio (SSMS)** da Microsoft.  
> Caso não tenha o SSMS instalado, não se preocupe! Incluí abaixo capturas de tela que mostram os resultados de cada exercício.

---

## 📚 Estrutura

As consultas foram desenvolvidas com base em um banco de dados contendo tabelas como:

- 🎞️ `Filmes`
- 👥 `Atores`
- 🧑‍🤝‍🧑 `ElencoFilme`
- 🏷️ `Generos`
- 🔗 `FilmesGenero`
